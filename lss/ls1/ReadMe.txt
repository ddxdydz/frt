Основные клавиши:
1) Открытие страницы в браузере CTRL + F5
2) Вывод окна с подсказками CTRL + Пробел
3) Для быстрого создания каркаса страницы используем ! + TAB
4) Для создания нескольих тэгов используйте конструкцию: имя тэга * количество тэгов,
например, h1 * 4, получим четыре парных тэга h1
5) Копировать элементы на странице - CTRL + C, а вставить скопированные элементы - CTRL + V
6) Для восстановления измененного(удаленного контента) используйте CTRL + Z
7) После внесения любых изменений на странице нажимайте CTRL + S - для сохранения информации
8) Для быстрого добавления/удаления комментариев используйте сочетание клавиш CTRL + /
